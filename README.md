# cub3d

For linux: 
`sudo apt install
  xorg openbox
  libxext-dev
  libssl-dev
  libbsd-dev`

`make ARGS=0.cub run`

![cub3d](https://raw.githubusercontent.com/untel/cub3d/master/pics/demo.png)

Bonus dones:
hud
win/lose hp on healthbar
weapon/fire
crosshair
jump/crouch
watch up/down
inc/dec speed
trigger collision
animated sprites
